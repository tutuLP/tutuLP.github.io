# 小说

小说灵感来源:佛家六道轮回,轮回乃是业力的相续与五蕴的刹那生灭,另一世的我并不是原来的我-无我学说.

生前所思、所言、所行都会在阿赖耶识（藏识）中留下“种子”，这些种子成熟时便形成果报，决定死后中阴身投向何道

十善业（不杀、不盗、不邪淫、不妄语等）对应三善道：天、人、阿修罗。

十恶业（杀、盗、淫、妄、两舌、恶口、绮语、贪、嗔、邪见）对应三恶道：地狱、饿鬼、畜生 

人在临终时最强的那一念（或善或恶）往往成为“先成熟”的业，直接决定中阴身最先见到的景象与投生处；随后其它业再渐次成熟，形成来生的具体命运.
 
 天:三界二十八天:四天王天、忉利天（帝释天）、夜摩天、兜率天、化乐天、他化自在天，以及色界、无色界诸天。这些天人具有光明身，寿命极长，享受妙乐，具有神通（如天眼、天耳等）。 当善业耗尽，仍会堕入其他道,甚至可能直落地狱 --天人死时，有五种衰相（衣裳垢腻、头上花萎、身体臭秽、腋下汗出、不乐本座），即知将死，随业转生

阿修罗:阿修罗身形高大，具神通，常与天人争斗,有的貌美（如天阿修罗），有的丑陋（如鬼阿修罗）.阿修罗因善业得神通，但嗔恨、傲慢心重.

地狱道:地狱众生受极苦，身形由业力化现，或如火焦，或如寒冰，无固定形态。佛教描述有八热地狱（如无间地狱）、八寒地狱等，受苦时间极长,《地藏经》云："业力甚大，能敌须弥，能深巨海，能障圣道。"

饿鬼道:饿鬼身形羸瘦，咽喉细如针，腹大如鼓，常受饥渴之苦。部分饿鬼有微通，能隐形或现形吓人。佛教分类中有"无财鬼"（不得食）、"少财鬼"（得少量食）等,饿鬼道善业极少，但恶业未尽时，可能长期滞留。善业增长（如得超度）可转生他道。

畜生道:佛教认为部分畜生有微通（如龙族可化人形），但多数愚痴，难修善法。





譬如《长阿含·世记经》说“地狱在南阎浮提之下五百由旬”，看似有坐标，实则指“向下”是“向下堕落”的隐喻；饿鬼道又被说在“人间之侧”，意思是“同处一界却彼此视而不见”。

- 欲界（含地狱、饿鬼、畜生、人、阿修罗、欲界六天）
- 色界（四禅十八天）
- 无色界（四空天）



- 业力相通就能交叉：天人可下凡与人间交合（佛经里“感生”故事），饿鬼可附人或托梦，地狱众生业尽即“弹出”转生他道。
- 佛陀、大阿罗汉也能“化现”任何一道，为彼说法，但这属于主动“调频”，不是它们自然看得见我们。



三界

一、欲界六天（仍存饮食、男女之欲）

1. 四天王天
2. 住户：四大天王（持国、增长、广目、多闻）及其部众——夜叉、乾闼婆、富单那等。
3. 高度：须弥山腰，与人界接壤。
4. 忉利天（又称三十三天）
5. 住户：帝释天（因陀罗）为天主，三十三位天王辅政，皆属“地居天”。
6. 高度：须弥山顶，再往上就“凌空”了。
7. 夜摩天
8. 住户：夜摩天王及“空居天”众。此处以上不再依托山地，而是“云居”。
9. 特点：日月光明已不及，天人自身放光。
10. 兜率天
11. 住户：
    - 内院：候补佛——现由弥勒菩萨住持，将来下生成佛。
    - 外院：一般欲界天人。
    - 特点：佛陀之母摩耶夫人死后先升此天，佛陀成道后上升内院为母说法。
12. 化乐天
13. 住户：化乐天王及众。
14. 特点：想娱乐时，只需动念，五欲器物“化现”自玩，不需外求。
15. 他化自在天
16. 住户：他化自在天王（佛教称“魔波旬”即居此）。
17. 特点：不用自己“化乐”，直接“拿”下层天的现成娱乐，故名“他化”。
18. 角色：常被视为“修道外魔”源头，但亦属轮回众生，福报尽仍堕落。



二、色界（已离欲，但仍有“色身”——微细光体）

色界分“四禅天”，每禅又开若干层，共十八天（一说十六、十七）。住户全是修“色界定”成就者，命终后“生到”对应频道。

初禅三天

1. 梵众天　2. 梵辅天　3. 大梵天
2. 住户：初禅成就者，大梵天王被视为“娑婆世界主”，自称造物主，但佛教认为其仍属轮回。

二禅三天

4. 少光天　5. 无量光天　6. 光音天

住户：二禅成就者，身放净光，以“光”为语言，故号光音天。

典故：光音天众生曾飞临地球，食地味后身重，遂成人类——这是佛教“地球人类起源说”。

三禅三天

7. 少净天　8. 无量净天　9. 遍净天

住户：三禅成就者，喜乐转淡，唯有“净乐”。

四禅九天（最复杂，含“五不还天”）

10. 无云天　11. 福生天　12. 广果天
11. 无烦天　14. 无热天　15. 善现天　16. 善见天　17. 色究竟天（阿迦尼吒天）

住户：

- 10–12层：一般四禅凡夫、外道。
- 13–17层：佛教三果圣人（阿那含）所居，又称“五不还天”——在此直接证涅槃，不再还回欲界。



三、无色界（连“色身”也舍掉，只剩“识”）

修“无色界定”者死后生此，纯是“意识层”，无物质、无方位、无光音，寿尽仍下落轮回。分四层：

1. 空无边处天　2. 识无边处天
2. 无所有处天　4. 非想非非想处天
3. 住户：对应四无色定的成就者，外道最高可入第4层，寿长八万大劫，但佛教认为仍未出世，定力退时还会堕落。
