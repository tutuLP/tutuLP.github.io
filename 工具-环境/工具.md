# 录像

mac中截图：
shift+command+4

mac中屏幕录制：现象中可以选择开启麦克风
shift+command+5

格式转化-压缩体积 1/10：

~~~shell
# 下载
brew install ffmpeg

# 常用
ffmpeg -i input.mov -c:v libx264 -c:a mp3 output.avi

# 牺牲质量-实际效果也很好
ffmpeg -i input.mov -c:v libx264 -preset fast -c:a mp3 output.avi

# 无损
ffmpeg -i input.mov -c copy output.avi

~~~
