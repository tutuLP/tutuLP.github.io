# 音乐

## 伴奏分离

```sh
# 创建虚拟环境
python3 -m venv spleeter-env
# 使用虚拟环境
source spleeter-env/bin/activate
# 安装
pip install spleeter
# 使用
spleeter separate -i your.mp3 -p spleeter:2stems -o output
```



Command + Shift + .  显示mac的隐藏文件夹
