# coding rules

> 整体思想:使用vibe coding,也就是ai辅助编程的思想

* 前端框架
Vue3 或 React
* 包管理工具
pnpm
* 样式
使用原生css不要使用sass语法，如果不是需要复用的样式优先使用 TailwindCSS
* 构建工具
Vite
* ui组件库
ElementPlus，AntDesign,Shadcn/UI
* 工程化工具
1. 使用 TypeScript,不能使用js
2. 使用 prettler 进行代码格式化
3. 使用 ESLint 进行规范检查
* 组件使用
1. Pinia 做状态管理
2. 使用 Axios 做 http 请求
3. Three.js 在网页上渲染3d场景
4. WebAssembly 网页性能天花板使用rust或c++写高性能模块编译后在浏览器上跑，比如视频剪辑图像处理等
* 其他开发工具
1. 使用 VitePress 搭建文档网站
2. 使用 Astro 进行静态站点生成,可以做博客类网站
* 跨平台
1. uni-app
2. Taro
3. ReactNative 跨平台
4. React 移动开发
5. Electron 跨平台桌面应用

## tailwindcss v4

`pnpm add -D tailwindcss @tailwindcss/cli postcss autoprefixer @tailwindcss/postcss`
`pnpm exec tailwindcss --help`

## prettler v3

`pnpm add -D prettier prettier-plugin-tailwindcss`
可以安装vscode安装插件:Prettier - Code formatter,在保存的时候自动fmt

## ESLint v9

`pnpm add -D eslint@^9 @eslint/js eslint-plugin-vue@^9 prettier eslint-config-prettier typescript-eslint`
可以安装vscode安装插件插件ESLintk
