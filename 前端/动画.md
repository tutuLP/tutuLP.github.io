# 滚动显示和隐藏菜单栏的方案

## 1

```
<div id="navbar">菜单内容</div>

#navbar {
  position: fixed;
  top: 0;
  width: 100%;
  transition: top 0.3s; /* 增加平滑过渡效果 */
}

aScript 逻辑：
记录上一次滚动位置 lastScrollTop。
在 window.onscroll 事件中比较当前滚动位置。
向下滚动：top: -50px (隐藏)；向上滚动：top: 0 (显示)。 

https://www.w3school.com.cn/howto/howto_js_navbar_hide_scroll.asp
```
